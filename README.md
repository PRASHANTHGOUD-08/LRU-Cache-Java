# LRU Cache (Java)

This project implements a Least Recently Used (LRU) Cache using Java.

## 🚀 Features
- O(1) time complexity for get and put
- Efficient cache eviction using Doubly Linked List
- HashMap for fast lookup

## 🧠 Concepts Used
- Data Structures
- HashMap
- Doubly Linked List

## 💻 How to Run
1. Compile the code
2. Run the program
3. Use menu options to test cache

## 📌 Example
- Insert elements
- Access elements
- Observe LRU eviction
